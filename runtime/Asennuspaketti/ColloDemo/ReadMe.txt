Asennat tämän Demo-softan täysin omalla vastuullasi.
Mitään takuita toimivuudesta ei myönnetä ja kaikki oikeudet pidätetään
Ongelma tapauksissa voit ottaa yhteyttä ColloidTekin asiakaspalveluun sähköpostilla
osoitteeseen roskapostia@colloidtek.fi

Asennusohjeet:

1. Poista vanha versiot siitä Demosta jota olet asentamassa
2. Pura ColloDemo.zip tiedosto
3. Asenna puretussa hakemistossa oleva .exe-tiedosto
	- Jos koneella ei ole ennestään Matlabin Runtime-ohjelmistoa se asennetaan
	  samalla
4. Aja puretussa hakemistossa oleva "ColloDemoSetup.bat"-tiedosto
5. Voit halutessasi poistaa puretun kansion ja zip-tiedoston
